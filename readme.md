# Perfumito Challenge – ML Importaciones

Juego interactivo educativo creado para **ML Importaciones**.  
Demuestra tus conocimientos sobre perfumes, zapatillas y atención personalizada y gana un **10% de descuento** real.

---

## 🎮 ¿Cómo funciona?

- Responde **5 preguntas** en **60 segundos**.
- Necesitas **5/5 respuestas correctas** para ganar.
- Al ganar, obtienes un **código exclusivo de 10% OFF** que se puede usar en tu próxima compra.

## ✨ Características

- Diseño premium oscuro con tonos púrpura y dorado
- Totalmente responsive (funciona excelente en celular)
- Feedback visual inmediato
- Código de descuento generado automáticamente
- Integración directa con WhatsApp
